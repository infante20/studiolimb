check_circle Pairing Principles

  • Contrast styles: geometric serif + humanist sans
  • One display font max — keep body neutral
  • Different weight classes for clear hierarchy
  • Test at small sizes — body fonts must be readable at 16px

tips_and_updates Tips

  • Load only the weights you actually use
  • 2 font families is the limit for web performance
  • Inter, Plus Jakarta Sans are safe neutral body fonts
  • Preview dark mode — some fonts look worse inverted
